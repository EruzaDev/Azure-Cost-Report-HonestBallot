# HonestBallot - Secure Voting Application

## 📌 Project Overview
HonestBallot is a secure and transparent cross-platform voting application developed using the Python Flet framework. The system supports multiple user roles including voters, politicians, COMELEC officials, and NBI officers, each with dedicated functionalities for managing elections, monitoring results, and handling candidate records.  

The application features secure authentication, real-time vote counting, audit logging, session management, and legal record tracking for candidates. Designed as a local-only platform using SQLite, HonestBallot can run both in a web browser and desktop environment across Windows, macOS, and Linux systems.
